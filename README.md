# BrickBridge.io - Global Building Block Industry Bridge

BrickBridge.io is a full-chain solution platform designed to connect global building block designers and enthusiasts with top-tier industrial manufacturing. 

## Core Mission
To serve as a high-fidelity "bridge" that eliminates the gap between design and physical fulfillment, optimizing the supply chain for both B2C and B2B sectors.

- **Direct Industrial ID Matching**: Native support for factory闆朵欢缂栫爜 (Industrial IDs like GDS), eliminating conversion errors.
- **Full-Chain Fulfillment**: No "blind zones" in parts; if it exists in the factory network, it's available.
- **Designer Profit Ecosystem**: Automated revenue sharing for signed designers through packaged part sets.
- **B2B Solution Portal**: Dedicated portals for distributors, wholesalers, and corporate customization clients.
- **ERP-Ready Architecture**: Built for direct integration with factory supply chain ERP systems.

## Tech Stack
- **Frontend**: Next.js 14+ (App Router), Tailwind CSS, TypeScript.
- **Backend**: API Routes (Next.js), planned middleware for ERP connectivity.
- **Infrastructure**: Designed for high-performance product indexing and complex part-list calculation.

## Project Structure
- `/app`: Frontend pages and API routes.
- `/components`: Reusable UI components.
- `/lib`: Core business logic, ERP connectors, and utility functions.
- `/types`: TypeScript definitions for Parts, MOCs, and Orders.
